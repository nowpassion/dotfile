## 비교: 수동 vs 스크립트

### 수동으로 할 경우 (15-20분)
```
1. mkdir ../ncpf
2. cd ../ncpf
3. git init
4. git branch -M main
5. cp ../dotfile/ncpf.nvim/lua . -r
6. cp ../dotfile/ncpf.nvim/plugin . -r
7. cp ../dotfile/ncpf.nvim/doc . -r
8. ... (15개 파일 각각 복사)
9. git add .
10. git commit -m "..."
11. git tag ...
12. GitHub 설정
13. git remote add ...
14. git push ...
```

### 스크립트 사용 (1-2분)
```
1. ./create-ncpf-repo.sh
2. GitHub 리파지토리 생성
3. git push (스크립트가 알려준 명령어 실행)
```

## 체크리스트

### 스크립트 실행 전
- [ ] dotfile 리파지토리 루트에 있는지 확인
- [ ] ncpf.nvim 디렉토리가 존재하는지 확인
- [ ] ../ncpf 디렉토리가 없는지 확인 (또는 덮어쓸 준비)

### 스크립트 실행 중
- [ ] 에러 메시지가 없는지 확인
- [ ] "✅ Repository created successfully!" 메시지 확인

### 스크립트 실행 후
- [ ] cd ../ncpf로 이동 가능한지 확인
- [ ] git log로 커밋 확인
- [ ] git tag로 v1.0.0 확인
- [ ] ls로 모든 파일 존재 확인

### GitHub 업로드 전
- [ ] GitHub 계정 로그인 확인
- [ ] SSH 키 또는 Personal Access Token 준비

### GitHub 업로드 후
- [ ] GitHub에서 파일들이 보이는지 확인
- [ ] README가 제대로 렌더링되는지 확인
- [ ] Releases에 v1.0.0이 보이는지 확인
